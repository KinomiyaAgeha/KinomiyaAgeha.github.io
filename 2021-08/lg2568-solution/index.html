设 p\large pp 为质数且 p≤n\large p\le np≤n。

显然的，若 gcd(x,y)=1\large \gcd(x,y)=1gcd(x,y)=1，则 gcd(x×p,y×p)=p\large \gcd(x \times p,y \times p)=pgcd(x×p,y×p)=p。

问题转化为求互质的数对 (x,y)\large (x,y)(x,y) 的个数。

这时候就要用上大毒瘤欧拉的欧拉函数了！

由于欧拉函数是与一个数互质，那么用前缀和就好了。

由于 (x,y)\large (x,y)(x,y) 与 (y,x)\large (y,x)(y,x) 算两种，所以计数时要乘2，但是会多加一次 ϕ(1)\large \phi(1)ϕ(1)，减去就好了。

设 m\large mm 为 n\large nn 的约数个数，pi\large p_ip​i​​ 为 n\large nn 的第 i\large ii 个约数。

则答案为

∑i=1m2×ϕ(npi)−ϕ(1)\large \sum_{i=1}^m{2 \times \phi(\frac{n}{p_i})-\phi(1)} ​i=1​∑​m​​2×ϕ(​p​i​​​​n​​)−ϕ(1)

2×∑i=1mϕ(npi)−1\large 2 \times \sum_{i=1}^m{\phi(\frac{n}{p_i})-1} 2×​i=1​∑​m​​ϕ(​p​i​​​​n​​)−1

2×(∑i=1mϕ(npi))−m\large 2 \times (\sum_{i=1}^m{\phi(\frac{n}{p_i})}) - m 2×(​i=1​∑​m​​ϕ(​p​i​​​​n​​))−m

实现的时候用欧拉筛，更加毒瘤了

#include<cstdio>
#include<iostream>
using namespace std;
#define ll long long
const int N=1e7+6;
ll n, m, ans, v[N], p[N], phi[N];
int main() {
    int i, j;
    scanf("%lld",&n);
    phi[1]=1;
    for(i=2;i<=n;++i) {
        if(!v[i]) v[i]=i, p[++m]=i, phi[i]=i-1;
        for(j=1;j<=m;++j) {
            if(p[j]>v[i]||p[j]*i>n) break ;
            v[p[j]*i]=p[j];
            phi[p[j]*i]=phi[i]*(i%p[j]?p[j]-1:p[j]);
        }
        phi[i]+=phi[i-1];
    }
    for(i=1;i<=m;++i) ans+=phi[n/p[i]];
    printf("%lld\n",(ans<<1)-m);
}
