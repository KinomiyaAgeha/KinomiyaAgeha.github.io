CF1083E The Fair Nut and Rectangles 题解

发表于 2021-09-21 更新于 2021-12-19 分类于 题解 阅读次数：

考虑 DP。

显然 DP 需要一定的顺序。

因为给出的矩形没有包含的关系，所以我们按照每个矩形右上角点的横坐标 xxx 递增排序，那么纵坐标 y\large yy 一定是递减排序的，反之亦然。

设 Si=xi×yi\large S_i = x_i \times y_iSi​=xi​×yi​。

因为每个矩形都有选与不选两种选择，所以状态为

设 f(i)\large f(i)f(i) 为选择第 iii 和矩形获得的最大收益。

初始值为 f(i)=Si−ai\large f(i) = S_i-a_if(i)=Si​−ai​

显然，转移为

f(i)=max⁡1≤j<i{f(j)+Si−ai−Si∪Sj}\large f(i)=\max_{1 \le j < i}{ \{ f(j)+ S_i - a_i - S_i \cup S_j \} } f(i)=1≤j<imax​{f(j)+Si​−ai​−Si​∪Sj​}

答案为

max⁡1≤i≤n{f(i)}\large \max_{1 \le i \le n}\{ f(i) \} 1≤i≤nmax​{f(i)}

复杂度为 O(n2)\large O(n^2)O(n2)。

 

显然，复杂度过高，考虑优化。

由于我们已经将矩形排序，所以

∀Si∪Sjxi≥xj,yi≤yj\large \forall S_i \cup S_j \quad x_i \ge x_j,y_i \le y_j ∀Si​∪Sj​xi​≥xj​,yi​≤yj​

即

Si∪Sj=xj×yi\large S_i \cup S_j = x_j \times y_i Si​∪Sj​=xj​×yi​

所以原方程可化简为

f(i)=max⁡1≤j<i{f(j)+xiyi−ai−xjyi}\large f(i) = \max_{ 1 \le j < i } { \{ f(j)+ x_iy_i - a_i -x_jy_i \} } f(i)=1≤j<imax​{f(j)+xi​yi​−ai​−xj​yi​}

按照套路去掉 max⁡\large \maxmax 函数，移项得

f(j)=yixj+f(i)−xiyi+ai\large f(j)=y_ix_j +f(i) - x_iy_i + a_i f(j)=yi​xj​+f(i)−xi​yi​+ai​

即

{y=f(j)k=yix=xjb=f(i)−xiyi+ai\begin{cases} \large y=f(j) \\ \large k=y_i \\ \large x=x_j \\ \large b= f(i)-x_iy_i+a_i \end{cases} ⎩⎪⎪⎪⎨⎪⎪⎪⎧​y=f(j)k=yi​x=xj​b=f(i)−xi​yi​+ai​​

对应到坐标系里即为：

每个决策点为 (xj,f(j))\large (x_j,f(j))(xj​,f(j))，其斜率 yi\large y_iyi​ 单调递减，所以要维护一个下凸壳。

由于斜率 yi\large y_iyi​ 单调递减 ，所以合法决策的斜率一定小于 yi\large y_iyi​。

#include<cstdio>
#include<iostream>
#include<algorithm>
using namespace std;
#define ll long long
#define X(y) (w[y].x)
#define Y(x) (f[x])
const int N=1e6+6;
ll n, ans, q[N], f[N];
struct Squ { ll x, y, z; } w[N];
bool operator<(Squ a,Squ b) { return a.y>b.y; }
double calc(ll x,ll y) { return X(x)!=X(y)? 1.0*(Y(x)-Y(y))/(X(x)-X(y)):1e9; }
int main() {
    int i;
    scanf("%lld",&n);
    for(i=1;i<=n;++i) scanf("%lld%lld%lld",&w[i].x,&w[i].y,&w[i].z);
    sort(w+1,w+n+1);
    int l=1, r=1;
    for(i=1;i<=n;++i) {
        f[i]=w[i].x*w[i].y-w[i].z;
        while(l<r&&calc(q[l],q[l+1])>=w[i].y) ++l;
        f[i]=max(f[i],f[q[l]]+(w[i].x-w[q[l]].x)*w[i].y-w[i].z);
        ans=max(ans,f[i]);
        while(l<r&&calc(q[r-1],q[r])<=calc(q[r],i)) --r;
        q[++r]=i;
    }
    printf("%lld\n",ans);
}
