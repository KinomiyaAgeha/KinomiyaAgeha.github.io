luogu5008 锦鲤抄 题解

发表于 2021-09-20 更新于 2021-12-19 分类于 题解 阅读次数：

首先考虑 DAG 的情况。

拓扑序实际上也是「无后效性」的体现，即若存在 (x→y)\large (x \rightarrow y)(x→y)，xxx 无论怎样处理都不会影响到 y\large yy。

同理，逆拓扑序也可以理解为无论 y\large yy 怎样处理，都不会影响 x\large xx。

在本题中，可以选择入度不为 0 但可以没有出边的点.也就是如果删去 yyy 和它的一条出边，不会使 x\large xx 受到影响，关于 x\large xx 的任何操作结果也不会因此变差。

所以只要优先删去权值最大的点就行了，最终会留下一个入度为 0 的点。

我们只要用 Tarjan 算法求出 SCC 并且缩点，就得到了一个 DAG。

 

接下来讨论每个 SCC 内部的选择。

若存在入度为不为 0 的 SCC，设其为 x\large xx。

那么 x\large xx 内的点可以随便选。

证明：

如果 x\large xx 的入度不为 0，分以下情况讨论。

对于 x\large xx 中只有 1 个点的情况，显然成立。

否则，x\large xx 至少由一个简单环构成，且一定存在入度 ≥2\large \ge 2≥2 的点，设其为 uuu。

对于 x\large xx 中的每个简单环，设其入度最大的点为 u\large uu，则至少可以删去 1 条 u\large uu 的入边，断开这个简单环.而只要 u\large uu 的入度不为 0，就不会影响到点的选择。最终可以得到一个多了若干条入边的 DAG。

因为 x\large xx 入度不为 0，所以这个 DAG 不存在入度为 0 的点，命题得证。

考虑入度为 0 的 SCC，设其为 y\large yy。

仿照上述证明思路，我们仍以 y\large yy 内每个简单环删去若干边为代价，得到一个 DAG。

不难发现，它是一个普通 DAG，而我们不能选择删去一个入度为 0 的点，设其为 v\large vv。

所以最优解为 v\large vv 的点权为 y\large yy 中最小。

更进一步地说，以上关于 SCC 的删边策略，最终得到的都是 SCC 的搜索树，搜索树一定是 DAG。

不同的是前者没有入度为 0 的点，后者有且仅有一个。

 

所以我们按照上述操作之后，把能够选择的点集（决策集合）排个序，取前 k\large kk 大就可以。

求出 SCC 的顺序即为逆拓扑序。

实现不再赘述。

updata 2021.9.24 修改了证明部分的漏洞

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
#include<vector>
using namespace std;
const int N=5e5+6, M=2e6+6;
int n, m, k, num, tp, dfn[N], low[N], st[N];
int scc, o, c[N], w[N], deg[N], f[N], ans[N];
int cnt, h[N], ver[M], nxt[M];
vector<int> p[N];
void add(int x,int y) { ver[++cnt]=y, nxt[cnt]=h[x], h[x]=cnt; }
void tarjan(int x) {
    int i, y;
    dfn[x]=low[x]=++num, st[++tp]=x;
    for(i=h[x];i;i=nxt[i]) {
        y=ver[i];
        if(!dfn[y]) {
            tarjan(y);
            low[x]=min(low[x],low[y]);
        } else if(!c[y]) low[x]=min(low[x],dfn[y]);
    }
    if(dfn[x]==low[x]) {
        ++scc;
        do {
            y=st[tp--], c[y]=scc, f[scc]=min(f[scc],w[y]);
            p[scc].push_back(y);
        } while(x!=y);
    }
}
void sol() {
    int i, x, y;
    for(x=1;x<=n;++x) for(i=h[x];i;i=nxt[i]) {
        y=ver[i];
        if(c[x]!=c[y]) ++deg[c[y]];
    }
    for(x=1;x<=scc;++x) {
        bool fg=(deg[x]==0);
        for(i=0;i<p[x].size();++i) {
            y=p[x][i];
            if(w[y]==f[x]&&fg) fg=0; else ans[++o]=w[y];
        }
    }
}
int main() {
    int i, x, y, z=0;
    memset(f,0x3f,sizeof(f));
    scanf("%d%d%d",&n,&m,&k);
    for(i=1;i<=n;++i) scanf("%d",&w[i]);
    for(i=1;i<=m;++i) scanf("%d%d",&x,&y), add(x,y);
    for(i=1;i<=n;++i) if(!dfn[i]) tarjan(i);
    sol();
    if(o<=k) for(i=1;i<=o;++i) z+=ans[i];
    else { sort(ans+1,ans+o+1); for(i=o-k+1;i<=o;++i) z+=ans[i]; }
    printf("%d\n",z);
}
