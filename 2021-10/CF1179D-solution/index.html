CF1179D Fedor Runs for President 题解

发表于 2021-10-30 更新于 2021-12-19 分类于 题解 阅读次数：

树上任意两点间，有且仅有一条简单路径。

数量

Cn2=n(n−1)2\large C^2_n = \frac{n (n-1)}{2} Cn2​=2n(n−1)​

对于添加边 (x,y)\large (x,y)(x,y)，设它们树上路径一点为 z\large zz。

加边后显然会多出一个简单环

设为 Sz\large S_zSz​ 为以 z\large zz 为根且不含环上节点的子树大小。

则贡献为

Sz(n−Sz)2\large \frac{S_z(n-S_z)}{2} 2Sz​(n−Sz​)​

总贡献

∑z∈(x,y)Sz(n−Sz)2=∑z∈(x,y)(Szn−Sz2)2=n⋅∑z∈(x,y)Sz−∑z∈(x,y)Sz22\large \frac{\sum_{z \in (x,y)} S_z(n-S_z)}{2} \\ \large = \frac{\sum_{z \in (x,y)} (S_zn-S_z^2)}{2} \\ \large = \frac{n \cdot \sum_{z \in (x,y)} S_z - \sum_{z \in (x,y) } S_z^2 }{2} 2∑z∈(x,y)​Sz​(n−Sz​)​=2∑z∈(x,y)​(Sz​n−Sz2​)​=2n⋅∑z∈(x,y)​Sz​−∑z∈(x,y)​Sz2​​

考虑我们对 S\large SS 的定义中，只含一个路径上的点。由于这条限制，所以

∀z1,z2∈(x,y)Sz1∩Sz2=ϕ\large \forall z_1,z_2 \in (x,y) \quad S_{z_1} \cap S_{z_2} = \phi ∀z1​,z2​∈(x,y)Sz1​​∩Sz2​​=ϕ

所以得到总贡献

∑z∈(x,y)Sz=n⟹n2−∑z∈(x,y)Sz22\large \sum_{z \in (x,y)} S_z = n \Longrightarrow \frac{n^2 - \sum_{z \in (x,y) } S_z^2 }{2} z∈(x,y)∑​Sz​=n⟹2n2−∑z∈(x,y)​Sz2​​

问题转化为最小化 ∑z∈(x,y)Sz2=Δ\large \sum_{z \in (x,y) } S_z^2 = \Delta∑z∈(x,y)​Sz2​=Δ，可以 DP。

 

先考虑 Sz\large S_zSz​ 如何求出。

设 szx\large sz_xszx​ 以 x\large xx 为根的子树大小，画图不难发现

Sy=szx−szyy∈son(x)\large S_y = sz_x-sz_y \quad y \in son(x) Sy​=szx​−szy​y∈son(x)

懒得证了，挺显然的。

设 fx\large f_xfx​ 为 添加路径是 x\large xx 到以 x\large xx 为根子树内任意一点，满足上式最小的贡献。

fx=min⁡y∈son(x){fy+(szx−szy)}\large f_x = \min_{y \in son(x)} { \{ f_y + (sz_x-sz_y) \} } fx​=y∈son(x)min​{fy​+(szx​−szy​)}

 

这样就可以计算 Δ\large \DeltaΔ 了。

Δ=min⁡y,z∈son(x){fy+fz+(n−szy−szz)2}\large \Delta = \min_{y,z \in son(x)} { \{ f_y+f_z+(n-sz_y-sz_z)^2 \}} Δ=y,z∈son(x)min​{fy​+fz​+(n−szy​−szz​)2}

显然可以斜率优化，但问题在于有两个需要维护的值。

不难想到，我们贪心地选择 y\large yy，维护最优的 z\large zz。

然后就是套路推式子。

但是个人感觉本题并不容易将转移变形成一次函数的形式，直接套两个决策 z1,z2\large z_1,z_2z1​,z2​ 就行，最后推出

(fz1+szz12)−(fz2+szz22)szz1−szz2≤2⋅(n−szy)\large \frac{(f_{z_1}+sz_{z_1}^2)-(f_{z_2}+sz_{z_2}^2)}{sz_{z_1}-sz_{z_2}} \le 2 \cdot (n-sz_y) szz1​​−szz2​​(fz1​​+szz1​2​)−(fz2​​+szz2​2​)​≤2⋅(n−szy​)

不难发现，如果 szy\large sz_yszy​ 单调增，那么对应的直线的斜率单调减，所以按这个排序。

对于两个决策 k1,k2\large k_1,k_2k1​,k2​，且前者优于后者，那么一定有

fk1+szk12szk1≤fk2+szk22szk2\large \frac{f_{k_1}+sz_{k_1}^2}{sz_{k_1}} \le \frac{f_{k_2}+sz_{k_2}^2}{sz_{k_2}} szk1​​fk1​​+szk1​2​​≤szk2​​fk2​​+szk2​2​​

斜率是单调减的，决策集合就是一个上凸壳，单调队列维护即可。

 

最后讨论一下复杂度。

并不好表达，因为排序和斜率优化都是与子节点个数成正比的。

考虑每个点子节点个数的平均值 ξ\large \xiξ。

这个字母能表示随机变量，直接拿来用了（

如果树是一条链，那么每个点的子节点和为

∑i=1n−1i=Cn2=n(n−1)2\large \sum_{i=1}^{n-1} i = C_n^2 = \frac{n(n-1)}{2} i=1∑n−1​i=Cn2​=2n(n−1)​

则

ξ1=n−12\large \xi_1 = \frac{n-1}{2} ξ1​=2n−1​

如果树是一朵菊花，那么只有根有 n−1\large n-1n−1 个子节点。

ξ2=n−1n\large \xi_2 = \frac{n-1}{n} ξ2​=nn−1​

接着考虑普通树。

对于一颗普通树，不能产生贡献的叶子节点一定多于链，少于菊花。减少根的直接子节点的个数（其实就是叶子节点个数）一定会导致子节点个数和增大，反之只会减小。

所以，对于普通树

ξ∈(ξ2,ξ1)\large \xi \in (\xi_2,\xi_1) ξ∈(ξ2​,ξ1​)

总复杂度 O(nξn2log⁡2ξn)\large O(n \xi_n^2 \log_2 \xi_n)O(nξn2​log2​ξn​)

实际上挺快的。

#include<cstdio>
#include<iostream>
#include<algorithm>
using namespace std;
#define ll long long
const int N=5e5+6;
ll n, dlt=(1ll<<60), sz[N], p[N], q[N], f[N];
ll c, h[N], ver[N<<1], nxt[N<<1];
inline void add(ll x,ll y) { ver[++c]=y, nxt[c]=h[x], h[x]=c; }
inline bool cmp(ll x,ll y) { return sz[x]<sz[y]; }
inline ll squ(ll x) { return x*x; }
inline ll Min(ll x,ll y) { return x<y? x:y; }
inline ll X(ll x) { return sz[x]; }
inline ll Y(ll x) { return f[x]+squ(sz[x]); }
inline double calc(ll x,ll y) {
    if(X(x)!=X(y)) return 1.0*(Y(x)-Y(y))/(X(x)-X(y));
    return Y(x)>Y(y)? 1e9:-1e9;
    // 横坐标相同，那么如果 x 纵坐标更大，则其斜率更优，保留。反之舍去
}
void dp(ll x,ll fr) {
    ll i, y, cnt=0, l=1, r=0;
    sz[x]=1;
    for(i=h[x],y;i;i=nxt[i]) if(ver[i]!=fr) {
        y=ver[i];
        dp(y,x), sz[x]+=sz[y];
    }
    f[x]=squ(sz[x]);
    if(sz[x]==1) return;
    for(i=h[x];i;i=nxt[i]) if(ver[i]!=fr) {
        y=ver[i], p[++cnt]=y;
        f[x]=Min(f[x],f[y]+squ(sz[x]-sz[y]));
    }
    dlt=Min(dlt,f[x]+squ(n-sz[x]));
    sort(p+1,p+cnt+1,cmp), q[++r]=p[1]; // 最小的一定是决策之一
    for(i=2;i<=cnt;++i) {
        y=p[i];
        while(l<r&&calc(q[l+1],q[l])<=2.0*(n-sz[y])) ++l;
        dlt=Min(dlt,f[y]+f[q[l]]+squ(n-sz[q[l]]-sz[y]));
        while(l<r&&calc(y,q[r-1])<=calc(q[r],q[r-1])) --r;
        q[++r]=y;
        // 一定都是小于等于，不然 WA
    }
}
int main() {
    scanf("%lld",&n);
    for(ll i=1,x,y;i<n;++i) scanf("%lld%lld",&x,&y), add(x,y), add(y,x);
    dp(1,0);
    printf("%lld\n",n*(n-1)/2+(n*n-dlt)/2);
}
