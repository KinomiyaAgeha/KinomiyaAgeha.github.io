luogu2491 消防 题解

发表于 2021-10-01 更新于 2021-10-07 分类于 题解 阅读次数：

实际上就是 树网的核 的数据加强版。

原题暴力枚举即可，本题也可以用复杂度为 O(nlogSUM)O(n\log SUM)O(nlogSUM) 的二分答案，这里只讲述单调队列的 O(n)O(n)O(n) 算法。

题意：在树的直径上选择两个距离不超过 s\large ss 的点，最小化「偏心距」。

「偏心距」：树中距离直径最远的节点到直径的距离。

显然，可以用单调队列维护。

设直径为 u\large uu，其节点数为 o\large oo，直径上两点为 (ui,uj)\large (u_i,u_j)(u​i​​,u​j​​)，f(x)\large f(x)f(x) 为从 x\large xx 出发，不经过直径上的其他点，能够到达的最远距离，dis(x,y)\large dis(x,y)dis(x,y) 为 (x,y)\large (x,y)(x,y) 之间的距离。

不难想到此时偏心距为

maxi,j∈[1,s](maxi≤k≤j{f(k)},dis(u1,ui),dis(uj,us))\large \max_{i,j \in [1,s]} {(\max_{i \le k \le j} { \{ f(k) \},dis(u_1,u_i),dis(u_j,u_s))} } ​i,j∈[1,s]​max​​(​i≤k≤j​max​​{f(k)},dis(u​1​​,u​i​​),dis(u​j​​,u​s​​))

仔细观察不难发现，f(k)\large f(k)f(k) 的定义域为 [1,s]\large [1,s][1,s]，而无论如何选择两个点，最终一定取到max1≤k≤sf(k)\large \max_{1\le k \le s}{f(k)}max​1≤k≤s​​f(k)。

所以上式可化为

maxi,j∈[1,s](max1≤k≤s{f(k)},dis(u1,ui),dis(uj,uo))\large \max_{i,j \in [1,s]} {(\max_{1 \le k \le s}{\{ f(k) \}},dis(u_1,u_i),dis(u_j,u_o))} ​i,j∈[1,s]​max​​(​1≤k≤s​max​​{f(k)},dis(u​1​​,u​i​​),dis(u​j​​,u​o​​))

而它是一个定值，设其为 K\large KK。

求出 u\large uu 上节点距离的前缀和数组 SSS。

则

maxi,j∈[1,s](K,max{S(i),S(o)−S(j)})\large \max_{i,j \in [1,s]} { (K,\max \{ S(i),S(o)-S(j)\}}) ​i,j∈[1,s]​max​​(K,max{S(i),S(o)−S(j)})

用单调队列维护队头为 满足 S(j)−S(i)≤s\large S(j)-S(i) \le sS(j)−S(i)≤s 的最大的 j\large jj 即可。

#include<cstdio>
#include<iostream>
using namespace std;
#define R register
const int N=3e5+10;
int n, s, t, mx, o, d[N], pre[N], f[N], sum[N], a[N], b[N];
int c=1, h[N], ver[N<<1], w[N<<1], nxt[N<<1];
bool v[N];
void add(int x,int y,int z) { ver[++c]=y, w[c]=z, nxt[c]=h[x], h[x]=c; }
void DFS(int x,int fr) {
    R int i, y, z;
    for(i=h[x];i;i=nxt[i]) if(ver[i]!=fr) {
        y=ver[i], z=w[i], pre[y]=i, d[y]=d[x]+z;
        if(d[y]>mx) mx=d[y], o=y;
        DFS(y,x);
    }
}
void dfs(int x) {
    R int i, y, z;
    v[x]=1;
    for(i=h[x];i;i=nxt[i]) if(!v[ver[i]]) {
        y=ver[i], z=w[i];
        dfs(y);
        f[x]=max(f[x],f[y]+z);
    }
}
int main() {
    R int i, j, x, y, z;
    R int l, r, p, q, k=0, ans=1<<30;
    scanf("%d%d",&n,&s);
    for(i=1;i<n;++i) {
        scanf("%d%d%d",&x,&y,&z);
        add(x,y,z), add(y,x,z);
    }
    DFS(1,0), p=o, mx=d[o]=0;
    DFS(p,0), q=o;
    while(q!=p) v[a[++t]=q]=1, b[t+1]=w[pre[q]], q=ver[pre[q]^1];
    v[a[++t]=p]=1;
    for(i=1;i<=t;++i) {
        dfs(a[i]);
        k=max(k,f[a[i]]);
        sum[i]=sum[i-1]+b[i];
    }
    for(l=1,r=1;l<=t;++l) {
        while(r<t&&sum[r+1]-sum[l]<=s) ++r;
        z=max(k,max(sum[l],sum[t]-sum[r]));
        ans=min(ans,z);
    }
    printf("%d\n",ans);
}
