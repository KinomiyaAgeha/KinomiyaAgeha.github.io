POJ3017 Cut the Sequence 题解

发表于 2021-10-10 更新于 2021-12-19 分类于 题解 阅读次数：

设 fi\large f_ifi​ 为把前 i\large ii 个数合法分成若干段，每段最大值之和的最小值。

fi=min⁡0≤j<i and ∑k=j+1iAk≤M{fj+max⁡j+1≤k≤i{Ak}}\large f_i = \min_{0 \le j < i \, and \, \sum_{k=j+1}^i A_k \le M} { \left \{ f_j + \max_{j+1 \le k \le i } \{ A_k \} \right \} } fi​=0≤j<iand∑k=j+1i​Ak​≤Mmin​{fj​+j+1≤k≤imax​{Ak​}}

显然是 O(n2)\large O(n^2)O(n2) 的，考虑优化。

若 j\large jj 可能为最优决策，则它一定满足

∑k=j+1iAk≤M 且 ∑k=jiAk>M即 j 是满足左式的最小决策\large \sum_{k=j+1}^i A_k \le M \text{ 且 } \sum_{k=j}^i A_k > M \\ \large \text{即 $\large j$ 是满足左式的最小决策} k=j+1∑i​Ak​≤M 且 k=j∑i​Ak​>M即 j 是满足左式的最小决策

或

Aj=max⁡j≤k≤i{Ak} 即 Aj 是 A 在 [j,i] 上的最大值 \large A_j = \max_{j \le k \le i}{ \{ A_k \} } \\ \large \text{ 即 $A_j$ 是 $\large A$ 在 $\large [j,i]$ 上的最大值 } Aj​=j≤k≤imax​{Ak​} 即 Aj​ 是 A 在 [j,i] 上的最大值 

 

证明：

反证法。 若最优决策 j\large jj 不满足上述两个条件，那么

max⁡j≤k≤i{Ak}=max⁡j+1≤k≤i{Ak}\large \max_{j \le k \le i}{ \{ A_k \} } = \max_{j+1 \le k \le i}{ \{ A_k \} } j≤k≤imax​{Ak​}=j+1≤k≤imax​{Ak​}

因为 f\large ff 有单调性，所以 fj−1≤fj\large f_{j-1} \le f_jfj−1​≤fj​

即

fj−1+max⁡j≤k≤i{Ak}≤fj+max⁡j+1≤k≤i{Ak}\large f_{j-1}+\max_{j \le k \le i}{ \{ A_k \} } \le f_j + \max_{j+1 \le k \le i}{ \{ A_k \} } fj−1​+j≤k≤imax​{Ak​}≤fj​+j+1≤k≤imax​{Ak​}

与 j\large jj 是最优决策矛盾，固假设不成立，命题得证。

 

对于第一个条件，可以直接预处理 ∑k=j+1iAk≤M\large \sum_{k=j+1}^i A_k \le M∑k=j+1i​Ak​≤M 的最小的 j\large jj，记为 ci\large c_ici​。

对于第二个条件，一个决策 j1\large j_1j1​ 无用，当且仅当存在 j1≤j2 且 Aj1≤Aj2\large j_1 \le j_2 \text{ 且 } A_{j_1} \le A_{j_2}j1​≤j2​ 且 Aj1​​≤Aj2​​，用单调队列排除 j≤ci\large j \le c_ij≤ci​ 的不合法决策，维护 Aj\large A_jAj​ 单调减即可。

但是 max⁡j+1≤k≤i{Ak}\large \max_{j+1 \le k \le i}{ \{ A_k \} }maxj+1≤k≤i​{Ak​} 这个东西没有单调性，队头仅仅是最优决策的一种可能。

考虑用平衡树维护，维护 fj+max⁡j+1≤k≤i{Ak}\large f_j + \max_{j+1 \le k \le i}{ \{ A_k \} }fj​+maxj+1≤k≤i​{Ak​} 的较小值。

仔细观察，不难发现，由于有第二个条件的限制，所以对于队列中每个决策 j\large jj

max⁡j+1≤k≤i{Ak}=Aqj+1\large \max_{j+1 \le k \le i}{ \{ A_k \} } = A_{q_{j+1}} j+1≤k≤imax​{Ak​}=Aqj+1​​

所以维护 $\large f_j + A_{j+1} $ 即可。

复杂度 O(nlog⁡2n)\large O(n\log_2n)O(nlog2​n)

#include<cstdio>
#include<iostream>
#include<set>
using namespace std;
const int N=1e5+6;
#define ll long long
int n, a[N], c[N], q[N];
ll m, sum, f[N];
multiset<ll> s;
int main() {
    scanf("%d%lld",&n,&m);
    for(int i=1;i<=n;++i) scanf("%d",&a[i]);
    for(int i=1,j=0;i<=n;++i) {
        for(sum+=a[i];sum>m;++j) sum-=a[j+1];
        c[i]=j;
    }
    int l=1, r=0;
    for(int i=1;i<=n;++i) {
        while(l<=r&&q[l]<=c[i]) s.erase(f[q[l]]+a[q[l+1]]), ++l;
        while(l<=r&&a[q[r]]<=a[i]) s.erase(f[q[r-1]]+a[q[r]]), --r;
        if(l<=r) s.insert(f[q[r]]+a[i]);
        q[++r]=i, f[i]=f[c[i]]+a[q[l]];
        if(s.size()) f[i]=min(f[i],*s.begin());
    }
    printf("%lld\n",f[n]);
}
