luogu3166 数三角形 题解

发表于2021-08-28|题解

link

显然的，在 的网格中任选三点的方案数为 还要减去三点共线的情况。

三点共线且沿矩形边的方案数

三点共线且不沿矩形边的方案数

设斜线 ，则其经过点数为 。

又因为是三点共线，所以有 种可能。

该斜线可以左右平移 次，上下平移 次，还有正反两种可能。

乘法原理。

实现的时候将 与 都加 1，因为这是坐标，而组合数计算的是点数，并且计算斜边时也要注意边界的变化。

#include<cstdio>
using namespace std;
#define ll long long
ll n, m, x, y, z;
ll gcd(ll x,ll y) { return y? gcd(y,x%y):x; }
int main() {
    scanf("%lld%lld",&n,&m), ++n, ++m;
    z=n*m*(n*m-1)*(n*m-2)/6;
    x=n*(n-1)*(n-2)/6*m;
    y=m*(m-1)*(m-2)/6*n;
    z=z-x-y;
    for(int i=1;i<n;++i) for(int j=1;j<m;++j) z-=(n-i)*(m-j)*(gcd(i,j)-1<<1);
    printf("%lld\n",z);
}
