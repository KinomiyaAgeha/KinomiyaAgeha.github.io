CF359B Permutation 题解

发表于2022-07-03|题解

分析

小清新构造题。

不难发现，对于一个有序序列，假设是 那么 把 提前， 发现得到的结果是 。

把 提前， 发现得到的结果是 。

把 提前，，而在左式中直接绝对值没了，对答案贡献 ，而右式则会让答案减少 。其他的都是一一对应的，一减就没了。那么把 提前的答案就是 。

直接把 提前，其他的顺序输出。

CODE

#include<bits/stdc++.h>
using namespace std;
int n, k;
int main() {
	scanf("%d%d",&n,&k), n<<=1;
	printf("%d ",k+1);
	for(int i=1;i<=k;++i) printf("%d ",i);
	for(int i=k+2;i<=n;++i) printf("%d%c",i," \n"[i==n]);
}
