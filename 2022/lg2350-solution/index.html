luogu2350 外星人 题解

发表于2022-03-17|题解

闲话

看到题面的第一眼 地铁、老人、手机。

这到底是个啥？

难道是初中课本里说的那样写？ 看样例，显然不是这样。 难道是导数？高阶导数确实这么写，但是欧拉函数怎么求导？

最后发现结果是 套娃 次。

这不就是迭代么？出题人写错了啊！正确写法加个括号。

分析

出题人都把 的标准分解给你了，不从这里下手简直是浑身难受啊。

首先明确 ，当且仅当 。

然后 BZOJ 题面上有 这么个东西。

懒得证明，仔细看一下就能发现是显然的。

从中不难看出，每一次迭代，每个质因子的次数都在不断变小，也可能会产生新的质因子。而我们的目的就是最终把他变成 1 或 2。

那么每一次迭代，最多能够将 1 个质因子变成 2，所以我们就要求出每个质因子在整个过程中，能够产生的 2 的个数。

如果质因子中没有 2，才考虑变成 1，否则就不划算了。不难发现只需要在上一步统计的基础上 +1 就行了。

那么怎么统计呢？

设 为 能够产生的 2 的数量。考虑递推计算。

  • 与 互质，所以 。
  • 若 与 互质，那么 与 在 中是相互独立的，互不影响，。

其他情况就不便于统计了。

这样用线性筛，就能在 的时间内求出答案。

最后对于每一个质因子 和次数 ，累加 就好了。

code

#include<cstdio>
#include<iostream>
using namespace std;
const int N=1e5+5;
int t, n, cnt, f[N], p[N];
bool v[N];
void pre() {
	f[1]=1;
	for(int i=2;i<=1e5;++i) {
		if(!v[i]) p[++cnt]=i, f[i]=f[i-1];
		for(int j=1;j<=cnt&&i*p[j]<=1e5;++j) {
			v[i*p[j]]=1;
			f[i*p[j]]=f[i]+f[p[j]];
			if(!(i%p[j])) break;
		} 
	}
}
int main() {
	pre();
	scanf("%d",&t);
	while(t--) {
		scanf("%d",&n);
		int x, y, ans=1;
		for(int i=1;i<=n;++i) {	
			scanf("%d%d",&x,&y);
			if(x==2) --ans;
			ans+=f[x]*y;
		}
		printf("%d\n",ans);
	}
}
