luogu7715 Shape 题解

发表于2022-11-12|题解

直接枚举 ，，对每行每列都做前缀和，可以得到 20pts。

枚举点不如枚举横杠，设 为在节点 ，同时往上下方向扩展的格数。对于一个 ，枚举 ，如果 和 构成一个横杠，那么贡献为 。

可以拿到 50pts。

枚举横杠又不如枚举极长横杠。

考虑对于每一行 ，直接处理两个黑色格子中间的部分，设为 ，那么贡献为 考虑如果将 递增排序，那么对于排名为 的 会贡献出 次。

于是乎这部分的复杂度为 ，可以通过。

关于 的预处理，要摒弃求前缀和暴力判断的做法，下面给出官方题解中的 做法。

CODE

const int N=2e3+5;
int n, m, ans, t[N], a[N][N], f[N][N];
int calc(int x,int l,int r) {
	if(l>r) return 0;
	int cnt=0, ans=0;
	for (int i=l;i<=r;++i) t[++cnt]=f[x][i];
	sort(t+1,t+cnt+1);
	for(int i=1;i<=cnt;++i) ans+=t[i]*(cnt-i);
	return ans;
}
signed main() {
	n=read(), m=read();
	for(int i=1;i<=n;++i) for(int j=1;j<=m;++j) {
		a[i][j]=read()? 0:1;
	}
	for(int j=1;j<=m;++j) {
		for(int i=1,p=-1;i<=n;++i) {
			if(!a[i][j]) p=-1; else ++p;
			f[i][j]=p;
		}
		for (int i=n,p=-1;i;--i){
			if (!a[i][j]) p=-1; else ++p;
			f[i][j]=min(f[i][j],p);
		}
	}
	for(int x=1;x<=n;++x) {
		int pre=1;
		for(int y=1;y<=m;++y) if(!a[x][y]) ans+=calc(x,pre,y-1), pre=y+1;
		ans+=calc(x,pre,m);
	}
	printf("%lld\n",ans);
}
