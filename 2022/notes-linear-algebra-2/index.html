高斯消元

很有用的小学奥数知识，但是在初中阶段基本都是解二元一次方程组，完全体现不出来它的强大之处。

但是你总不能用代码直接模拟解 元 1 次方程组吧，所以在 OI 中还是很有用处的，而且牵扯到很多线性代数里的概念。

进入正题。

线性方程组定义为由 个 元一次方程构成的方程组，举个例子 把每个未知数的系数提出来，组成一个 的矩阵，称为“系数矩阵”，加上右侧的常数，就成为了 "增广矩阵"。 求解这个方程组的步骤可以概括成对增广矩阵的三类操作。

  1. 用一个非 0 的数乘某一行。
  2. 用一个非 0 的数乘某一行，加到另一行上。
  3. 交换两行的位置。

此三子者，称为矩阵的“初等行变换”。对以上使用若干次初等行变换，可以得到 代表着

这个矩阵称为“阶梯形矩阵”，其系数矩阵部分称为“上三角矩阵”。通过这个形状大致就能明白什么意思了。

最后化简为 这样就能直接看出每一个未知数的值。这个矩阵称为“简化阶梯矩阵”，其系数矩阵部分为“对角矩阵”。依旧是感性理解。

通过初等行变换，把增广矩阵变为简化阶梯形矩阵，从而求解线性方正组的算法，就是高斯消元。

其思想为，对于每个未知数 ，找到一个 系数非 0，但是 的系数都是 0 的方程，运用初等行变换把其他方程的 项系数都变为 0。

还有一种情况是当某一行系数全为 0，常数不为 0 时，原方程组无解。

具体实现看注释。

const double eps=1e-6;
int n, cur; // cur表示当前的行
double m[N][N];
void xiaoyuan() {
	for(int j=0;j<n;++j) {
		int x;
		for(x=cur;x<n;++x) if(fabs(m[x][j])>eps) break;
        // 找到j这一列的非0元素
        // 没有就continue
		if(x==n) continue;
		for(int i=j;i<=n;++i) swap(m[x][i],m[cur][i]);
        // 把非0元素所在行与当前行交换
		for(int i=n;i>=j;--i) m[cur][i]/=m[cur][j];
        // 当前行同除，让主元系数化为1
		for(int i=0;i<n;++i) {
			if(i!=cur) for(int k=n;k>=j;--k) m[i][k]-=m[cur][k]*m[i][j];
        // 对其他行消元
		}
		++cur;
	}
	if(cur<n) {
        // cur<n，cur到n这几行一定全为0
		for(int i=cur;i<n;++i)
			if(fabs(m[i][n])>eps) { puts("-1"); return; }
        // 0=一个常数，无解
		puts("0");
        // 0=0，无数解
	} else for(int i=0;i<n;++i) printf("%.2lf\n",m[i][n]);
}
