「概率论学习笔记」#1 初步概念与公式

update 2022.2.19

增加了数学期望相关内容。

前言

本人就没怎么系统地学习过概率这部分，自此开坑……

本文内容为概率论入门概念与公式，题目什么的，开学后有时间再说吧。

关于概率论中”试验“的概念不再赘述。

概率

某个随机试验所有的可能结果称为样本空间，记为 \(S\)。\(S\) 的每个元素即为样本点，同时它的子集为「随机事件」，简称事件。

设两个事件为 \(A\) 与 \(B\)。则有

\(A \cup B=A+B\)，称为 \(A\) 与 \(B\) 的和事件，即 \(A\) 与 \(B\) 其中至少有一个发生。

\(A \cap B = AB\)，称为 \(A\) 与 \(B\) 的积事件，即 \(A\) 与 \(B\) 同时发生。

这两个等号意思是左右两边写法不同，实际相同。下面的等号就是取值了。

\(A \cap B = \varnothing\)，称 \(A\) 与 \(B\) 互斥，指 \(A\) 与 \(B\) 不会同时发生。

\(A \cup B = S\) 且 \(A \cap B = \varnothing\)，称 \(A\) 与 \(B\) 互为对立事件，即对于每次试验，\(A\) 与 \(B\) 必有且仅有一个会发生。

如果 \(A\) 是否发生对 \(B\) 是否发生毫无影响，称 \(A\) 与 \(B\) 为互相独立的事件。

 

在相同条件下进行 \(n\) 次试验，其中事件 \(A\) 发生了 \(N_a\) 次，称 \(\frac{N_a}{n}\) 为事件 \(A\) 的频率。在重复进行同一试验时，\(A\) 发生的频率总是接近某个常数，这个常数称为事件 \(A\) 的概率，记为 \(P(A)\)。\(A\) 不发生的概率为 \(P(\overline A) = 1-P(A)\)。

概率有如下性质。

  1. 非负性：\(P(A) \in [0,1]\)。

  2. 规范性：对于必然事件 \(S\)，\(P(A)=1\)，对于不可能事件 \(S'\)，\(P(S')=0\)。

  3. 容斥性：\(P(A \cup B) = P(A)+P(B)-P(A \cap B)\)。特别地，若 \(A\) 与 \(B\) 为互斥事件，那么 \(P(A \cup B)=P(A)+P(B)\)。

  4. 互斥事件的可加性：就像上条的特例那样。若 \(\{ A_{1 \sim n} \}\) 为互斥的 \(n\) 个事件，那么 \[ P(\bigcup_{i=1}^n A_i) = \sum_{i=1}^n P(A_i) \] 如果 \(A\) 与 \(B\) 是对立事件，那它们一定互斥，而 \(A \cup B = S\)，所以 \(P(A \cup B) = P(A) + P(B) = 1\)。

  5. 独立事件的可乘性：若 \(A\) 与 \(B\) 是互相独立的事件，那么 \(P(A \cap B) = P(AB) = P(A) \cdot P(B)\)。若 \(\{ A_{1 \sim n} \}\) 为互相独立的 \(n\) 个事件，那么 \[ P(\bigcap_{i=1}^n A_i) = \prod_{i=1}^n A_i \]

  6. 伯努利大数定理：在某次试验中 \(A\) 发生的概率为 \(p\)，不发生的概率为 \(q\)，则在 \(n\) 次试验中 \(A\) 发生至少 \(m\) 次的概率为 \((p+q)^n\) 展开式中 \(p^n\) 到 \(p^m q^{n-m}\) 各项之和。

    如果在一次试验中，\(A\) 发生的概率为 \(p\)，不发生的概率为 \(q\)，那么 \(n\) 次试验中 \(A\) 恰好发生 \(k\) 次的概率为 \[ P_n(k) = C^k_n \cdot p^k \cdot q^{n-k} \]

第 6 条好像在 OI 中没啥用。

OI 中大多研究古典概率。在古典概率中，构成 \(A\) 的事件有 \(a\) 个，不构成的有 \(b\) 个，那么 \(P(A) = \frac{a}{a+b}\)。其实就是另一种形式的组合计数（）。

条件概率

\(P(A | B)\) 表示在发生 \(B\) 的前提下发生 \(A\)。有 \(P(A|B) = \frac{P(AB)}{P(B)}\)。

贝叶斯公式： \[ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} \] 全概率公式：样本空间 \(S\) 分成若干个不相交的部分 \(\{ B_{1 \sim n} \}\)，则 \[ P(A) = \sum_{i=1}^{n} P(A|B_i) \cdot P(B_i) \] 全概率公式的关键是划分样本空间，做到不重不漏，并计算出每个划分下的概率，从而得到总概率。

数学期望

如果一个随机变量全部可能的取值只有有限个或可数无穷个，则称它为一个离散型随机变量。

一个离散型随机变量的期望是试验中每个结果的概率乘以其结果的总和。换言之，设 \(X\) 为一个离散型随机变量，取值为 \(\{ x_{1 \sim n}\}\)，相应概率为 \(\{ p_{1 \sim n} \}\)，那么期望为 \[ E(X) = \sum_{i=1}^n p_i x_i \] 比如一枚骰子，设点数为 \(X\)，则 \(P(X=1) \sim P(X=6)\) 均为 \(\frac{1}{6}\)，对应的期望为 \(E(X) = \sum_{i=1}^6 {i \times \frac{1}{6}} = 3.5\)。

数学期望的相关内容：

  1. 数学期望是线性函数，满足 \(E(aX+bY) = a \cdot E(X) + b \cdot E(Y)\)。对应骰子那个例子，\(E(X=1) \sim E(X=6)\) 的期望都是各自 \(X\) 的取值，也就是上面那个 \(i\)。那么可以直接把 \(\frac{1}{6}\) 提出来，即 \(E(X) = \frac{1+2+3+4+5+6}{6} = 3.5\)。

  2. 独立事件的可乘性。同概率。 \[ E(X \cap Y) = E(XY) = E(X) \cdot E(Y) \]

  3. 概率 \(P(X)=p\) 时，期望成功次数为 \(\frac{1}{p}\)。

  4. 全期望公式

\[ E(Y) = E(E(Y|X)) = \sum_{i=1}^n P(X=x_i) \cdot E(Y | X=x_i) \]

说实话，本人目前并不理解全期望公式。以后明白了再来补上。(´。＿。｀)
