一个简单应用 0-1 Trie

把int当作长度为 的字符串插入 Trie 中。

下面放板子，可以快速处理最大异或和的。

int tot, trie[N*31][2];
// [数字个数*数位][每一位不是0就是1]
void insert(int S) {
	int x=0;
	for(int i=31;i>=0;--i) {
		int a=(S>>i)&1;
		if(!trie[x][a]) trie[x][a]=++tot;
		x=trie[x][a];
	}
}
int find(int S) {
	int x=0, ans=0;
	for(int i=31;i>=0;--i) {
		int a=(S>>i)&1;
		if(trie[x][a^1]) ans+=(1<<i), x=trie[x][a^1];
        // 尽量往数字相反的地方走，这样异或值为1
		else x=trie[x][a];
	}
	return ans;
}

DFA 是个啥？

有限自动机 (FA)，确定有限状态自动机 (DFA)。

我们通常用确定有限状态自动机 (DFA) 解决大部分字符串的问题。

DFA 可以用一个 元组 表示，其中 为状态集, 为字符集，为转移函数，为起始状态，为终态集。

如何判断一个字符串是否能被一个 DFA 接受呢？一开始时，自动机在起始状态 ，每读入一个字符 后，状态转移到 ，其中 为当前状态。当整个字符串读完之后, 当且仅当 在终态集 中时，DFA 接受这个字符串。

是一种最简单的 DFA。
