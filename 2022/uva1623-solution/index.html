UVA1623 神龙喝水 题解

发表于2022-02-16|题解

分析

贪心。

维护一个集合 ，保存不下雨的日期。设 为第 个湖上一次下雨是第 天，起初全部为 0。

  1. 对于一个 ，直接把 加入 。

  2. 对于一个 ，设 ，在集合中查找不小于 的最小的日期。如果没有，无解。否则更新 ，记录答案，并在 中删去找到的日期。

正确性？如果一天下雨的时候，下雨的那个湖不能在那一天之前，最后一次下雨之后喝干，那么肯定就溢出。否则就一定不会溢出。

最优性？由于是按照时间顺序处理的，选择最小的喝水日期一定不劣，不选择最小的喝水日期一定不优。

就像下面，第 天 要下雨，如果选择 4，那么 天 要下雨的时候就没得选了。如果按照上面的贪心，则是有解的。

S={1,4}
g[i]=1, g[j]=5
a[u]=i, a[v]=j  u<v

就氵吧。

CODE

#include<cstdio>
#include<cstring>
#include<iostream>
#include<set>
using namespace std;
#define iter set<int>::iterator
const int N=1e6+6;
int t, n, m, a[N], f[N], g[N];
set<int> s;
iter p;
inline void sol() {
    memset(g,0,sizeof(g)), memset(f,0,sizeof(f));
    s.clear();
    scanf("%d%d",&n,&m);
    for(int i=1;i<=m;++i) scanf("%d",&a[i]);
    bool fg=1;
    for(int i=1;i<=m;++i) {
        if(!a[i]) s.insert(i);
        else {
            int d=a[i];
            p=s.lower_bound(g[d]);
            if(p==s.end()) { fg=0; break; }
            else g[d]=i, f[*p]=a[i], s.erase(p);
        }
    }
    if(!fg) puts("NO");
    else {
        puts("YES");
        for(int i=1;i<=m;++i) if(!a[i]) printf("%d ",f[i]);
        puts("");
    }
}
int main() { for(scanf("%d",&t);t--;sol()); }
