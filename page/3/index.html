关于OI，不知所言

发表于 2021-11-06 更新于 2021-12-19 分类于 简记 本文字数： 1.5k 阅读时长 ≈ 3 分钟

我认为我是一个真实的人。

我一直在追寻，我的「真物」。

我最讨厌口是心非的行为，总是不加掩饰地流露自己最真实的想法。

我总是毫不保留地对那些仅仅算是认识的「朋友」诉以奇怪的碎碎念。

我瞧不起恶性竞争的制造者与参与者。

我尽力选择自己所喜欢的，抵制不喜欢的。

我想保持自己的本色，自己的思想，自己的世界。成为那个遥远的自己。

 

我是一个虚伪的人。

所谓「真物」，不过是我逃避现实的借口罢了。

表露自己，等于被大多数人孤立。

极其依赖朋友，却总止步于普通朋友。

总想着改变无法改变的事。

选择权，也许仅仅是掌权者的恩赐。

那些东西，无数次地让我与自己渐行渐远。
